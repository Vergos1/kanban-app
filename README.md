## About

Kanban App is a task management board application with authentication. Users can register, log in via Google OAuth, and manage tasks on a kanban board. The project follows a Containers and Presentational Components architecture with clear separation of concerns.

## Pages

| Page | Description |
|---|---|
| **Home** | Landing page |
| **Login** | User login with Google OAuth |
| **Registration** | User registration form |
| **Board** | Kanban board with task management |

## Features

- **Google OAuth** — sign in with Google via `@react-oauth/google`
- **Registration & Login** — user authentication forms
- **Kanban Board** — task board with columns and cards
- **Redux Toolkit** — global state management with slices
- **Mantine UI** — forms, modals, date pickers and hooks
- **Axios** — API requests to backend
- **SCSS** — component-scoped styling with autoprefixer
- **Container / Presentational pattern** — clean architecture separation

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
